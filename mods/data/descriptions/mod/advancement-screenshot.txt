<p><span style="font-size:18px">Advancement Screenshot is a client-side mod which can automatically document the journey through a world without having to remember to take screenshots manually. Every time an advancement is achieved, a screenshot will be taken. The same code as the screenshot key (F2) is used, so the screenshots are located in the default folder. Works in multiplayer environments with the mod only on the client.<br><br>There is a 1 second delay in between getting the advancement and the actual screenshot. This allows some time for the message in the top right to appear.</span></p><p style="text-align:center"><br><br><img src="https://cdn.serilum.com/description/nameplate/dependencies.svg"
