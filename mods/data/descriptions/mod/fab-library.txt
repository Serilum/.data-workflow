<p><span style="font-size:16px">The Find A Block Library assists in finding a specific block and keeping track of its location in a world efficiently. Especially useful when multiple mods depend on a specific blockstate in the world.</span><br><br><br><span style="font-size:14px">See the source code in the <strong><a style="font-size:14px" href="https://github.com/ricksouth/FAB-Library" target="_blank" rel="nofollow">Github repository</a></strong>.</span><br><span style="font-size:14px">For more information on what the mod does and its public functions see the <strong><a style="font-size:14px" href="https://github.com/ricksouth/FAB-Library/wiki" target="_blank" rel="nofollow">Wiki</a></strong>.</span><br><br><br><strong><span style="text-decoration:underline">Configurable</span><br></strong><strong>blockCheckAroundEntitiesDelayMs</strong>&nbsp;(default = 30000, min 0, max 3600000): The delay of the is-there-a-block-around-check around entities in ms. Increasing this number can increase TPS if needed.<br><br><br><span style="font-size:24px"><strong>How to use the library in your project:<br></strong></span><em>This is only for developing new mods. If you'd like to use a mod that depends on FAB Library, just download it from this page.</em><br><span style="font-size:24px"><strong>1.</strong> </span>Add the following lines to your <strong>build.gradle</strong> file:</p><hr><hr><hr><p>repositories {<br>&nbsp; &nbsp; maven {<br>&nbsp; &nbsp; &nbsp; &nbsp; url = "https<a style="text-decoration:none;color:#000" rel="nofollow">://github</a>.com/ricksouth/serilum-forge-maven/raw/maven/"<br>&nbsp; &nbsp; }<br>}</p><p>dependencies {<br>&nbsp; &nbsp; runtimeOnly fg.deobf("com.natamus.fablibrary:fablibrary:VERSION")<br>&nbsp; &nbsp; compileOnly fg.deobf("com.natamus.fablibrary:fablibrary:VERSION")<br>}</p><p><br><br><span style="font-size:24px"><strong>2. </strong></span>Replace <strong>VERSION</strong> in <span style="text-decoration:underline">the previous lines</span> with <strong><em>mcversion-modversion</em></strong>, e.g.<em>&nbsp;1.14.4-1.1, or&nbsp;1.15.2-1.1:</em><br><em><strong>Partial Example:</strong> dependencies {</em><br><em>&nbsp; &nbsp; runtimeOnly fg.deobf("com.natamus.fablibrary:fablibrary:1.15.2-1.0")</em><em><br><br><br></em><span style="font-size:24px"><strong>3. </strong></span>After that refresh your workspace by entering the following <strong>command</strong> in your <strong>project folder</strong>:<em><br></em>$ gradlew cleanEclipse eclipse --refresh-dependencies<br><br><br><span style="font-size:24px"><strong>4. </strong></span>Add the following lines to the bottom of&nbsp;<strong>mods.toml</strong>&nbsp;in your <strong>META-INF</strong> folder, replace <em>YOURMODID</em> with your mod's id:<br>[[dependencies.YOURMODID]]<br>&nbsp; &nbsp; modId="fablibrary"<br>&nbsp; &nbsp; mandatory=true<br>&nbsp; &nbsp; versionRange="[1.1,)"<br>&nbsp; &nbsp; ordering="NONE"<br>&nbsp; &nbsp; side="BOTH"<br><br><br><span style="font-size:24px"><strong>5. </strong></span>When uploading the file to your mod project page on curseforge, make sure to select FAB Library as a dependent at the bottom:<br><img src="https://cdn.serilum.com/projects/fab-library/a.png" width="721" height="242"></p>
