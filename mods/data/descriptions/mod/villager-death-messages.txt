<p><span style="font-size:18px">Villager Death Messages is a minimalistic mod that broadcasts an informational death message whenever a villager passes away. The message displays the villager's name if custom, profession, cause of death and last location. The mod also mentions modded villagers.</span></p><p style="text-align:center"><br><br><img src="https://cdn.serilum.com/description/nameplate/dependencies.svg" alt="Dependencies" width="838"
