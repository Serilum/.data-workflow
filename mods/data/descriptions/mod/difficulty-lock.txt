<p><span style="font-size:18px">Difficulty Lock adds an easy way to configure the difficulty in all (new) worlds. Can be useful for modpacks, or just in your singleplayer experience if you have a default difficulty you always play with. There are config options to force the Hardcore gamemode, disable cheats, or prevent Creative mode from being selected.<br><br>The Create World Screen is correctly updated with the config settings. Also works on (dedicated) servers.</span></p><p style="text-align:center"><br><br><img src="https://cdn.serilum.com/description/nameplate/dependencies.svg" alt="Dependencies" width="838" height="64"></p><p><a href="https://curseforge.com/minecraft/mc-mods/collective" target="_blank" rel="nofollow"><img src="https://cdn.serilum.com/description/badges/dependencies/requires/collective.svg" alt="Requires Collective" width="251" height="49"></a><br><a href="https://curseforge.com/minecraft/mc-mods/serilums-customization-bundle" target="_blank" rel="nofollow"><img src="https://cdn.serilum.com/description/badges/dependencies/includes/customization-bundle.svg" alt="Included in Serilum's Customization Bundle" width="416" height="49"></a><br><img src="https://cdn.serilum.com/description/badges/environment/server.svg" alt="Environment: Server" width="220" height="49"></p><p style="text-align:center"><br><br><br><img src="https://cdn.serilum.com/description/nameplate/configurable.svg" alt="Configurable" width="838" height="64"></p><p><a href="https://serilum.com/configuration-guide" target="_blank" rel="nofollow"><img src="https://cdn.serilum.com/description/badges/configuration-guide.svg" alt="Configuration guide" width="220" height="49"></a></p><p>&nbsp;&nbsp;&nbsp;<span style="font-size:18px">All available config options:</span></p><div class="spoiler" style="margin-left:10px;padding-left:10px"><p><span style="font-size:12px"><em>The priority shows how the mod handles when multiple gamemodes are set to true in the config. E.g. if all are enabled all new worlds will be peaceful.</em></span><br><br><span style="font-size:14px"><strong>forceHardcoreMode</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Sets the world game mode to hardcode mode and ignores the other difficulty settings when enabled.</span><br><br><span style="font-size:14px"><strong>disableCreativeModeSelection</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Priority 1: Sets the difficulty in any world to peaceful when enabled.</span><br><br><span style="font-size:14px"><strong>forceEasy</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Priority 2: Sets the difficulty in any world to easy when enabled.</span><br><br><span style="font-size:14px"><strong>forceNormal</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Priority 3: Sets the difficulty in any world to normal when enabled.</span><br><br><span style="font-size:14px"><strong>forceHard</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default true</span><br><span style="font-size:13px">Priority 4: Sets the difficulty in any world to hard when enabled.</span><br><br><span style="font-size:14px"><strong>forceCheatsDisabled</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Whether the cheats option
