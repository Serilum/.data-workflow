Especially useful when multiple mods are trying to change the same entity.</span><br><br><br><span style="font-size:14px">See the source code in the <strong><a style="font-size:14px" href="https://github.com/ricksouth/SAM-Library" target="_blank" rel="nofollow">Github repository</a></strong>.</span><br><span style="font-size:14px">For more information on what the mod does and its public functions see the <strong><a style="font-size:14px" href="https://github.com/ricksouth/SAM-Library/wiki" target="_blank" rel="nofollow">Wiki</a></strong>.</span><br><br><br><br><span style="font-size:24px"><strong>How to use the library in your project:</strong></span><br><span style="font-size:24px"><strong>1.</strong> </span>Add the following lines to your <strong>build.gradle</strong> file:</p><hr><hr><hr><p>repositories {<br>&nbsp; &nbsp; maven {<br>&nbsp; &nbsp; &nbsp; &nbsp; url = "https<a style="text-decoration:none;color:#000" rel="nofollow">://github</a>.com/ricksouth/serilum-forge-maven/raw/maven/"<br>&nbsp; &nbsp; }<br>}</p><p>dependencies {<br>&nbsp; &nbsp; runtimeOnly fg.deobf("com.natamus.samlibrary:samlibrary:VERSION")<br>&nbsp; &nbsp; compileOnly fg.deobf("com.natamus.samlibrary:samlibrary:VERSION")<br>}</p><p><br><br><span style="font-size:24px"><strong>2. </strong></span>Replace <strong>VERSION</strong> in <span style="text-decoration:underline">the previous lines</span> with <strong><em>mcversion-modversion</em></strong>, e.g.<em>&nbsp;1.14.4-1.0, or&nbsp;1.15.2-1.0:</em><br><em><strong>Partial Example:</strong> dependencies {</em><br><em>&nbsp; &nbsp; runtimeOnly fg.deobf("com.natamus.samlibrary:samlibrary:1.15.2-1.0")</em><em><br><br><br></em><span style="font-size:24px"><strong>3. </strong></span>After that refresh your workspace by entering the following <strong>command</strong> in your <strong>project folder</strong>:<em><br></em>$ gradlew cleanEclipse eclipse --refresh-dependencies<br><br><br><span style="font-size:24px"><strong>4. </strong></span>Add the following lines to the bottom of&nbsp;<strong>mods.toml</strong>&nbsp;in your <strong>META-INF</strong> folder, replace <em>YOURMODID</em> with your mod's id:<br>[[dependencies.YOURMODID]]<br>&nbsp; &nbsp; modId="samlibrary"<br>&nbsp; &nbsp; mandatory=true<br>&nbsp; &nbsp; versionRange="[1.0,)"<br>&nbsp; &nbsp; ordering="NONE"<br>&nbsp; &nbsp; side="BOTH"<br><br><br><span style="font-size:24px"><strong>5. </strong></span>When uploading the file to your mod project page on curseforge, make sure to select SAM Library as a dependent at the bottom.</p><p><br>------------------<br><br><span style="font-size:24px"><strong>You may freely use this mod in any modpack, as long as the download remains hosted within the CurseForge ecosystem.</strong></span><br><br><span style="font-size:18px"><a style="font-size:18px;color:#008000" href="https://serilum.com/" target="_blank" rel="nofollow">Serilum.com</a> contains an overview and more information on all mods available.</span><br><br><span style="font-size:14px">Comments are disabled as I'm unable to keep track of all the separate pages on each mod.</span><span style="font-size:14px"><br>For issues, ideas, suggestions or anything else there is the&nbsp;<a style="font-size:14px;color:#008000" href="https://github.com/Serilum/.issue-tracker" target="_blank" rel="nofollow">Github repo</a>.
