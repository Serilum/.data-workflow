The default settings reduce file sizes by 30%, with barely any visible difference. There are however multiple config options available to increase that number, by for example reducing the amount of colours or gradients.<br><br>Another effective option is to reduce the screenshot dimensions while maintaining the aspect ratio.</span></p><p style="text-align:center"><br><br><img src="https://cdn.serilum.com/description/nameplate/dependencies.svg" alt="Dependencies" width="838" style="font-size:18px">All available config options:</span></p><div class="spoiler" style="margin-left:10px;padding-left:10px"><p><span style="font-size:14px"><strong>sendScreenshotCompressMessage</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Whether a message should be sent in the chat of how much the screenshot size was compressed.</span><br><br><span style="font-size:14px"><strong>keepOriginalScreenshot</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">If the original, uncompressed, screenshot should be saved in ./screenshots/original/.</span><br><br><span style="font-size:14px"><strong>compressionQuality</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default 0F, min 0F, max 1F</span><br><span style="font-size:13px">The quality of the compressed screenshot. 0 is the strongest compression, 1 the weakest.</span><br><br><span style="font-size:14px"><strong>screenshotResizePercentage</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default 100, min 1, max 100</span><br><span style="font-size:13px">By default on 100%, so disabled. Resize taken screenshots to a lower resolution, while maintaining the aspect ratio. A lower number, means a lower file size.</span><br><br><span style="font-size:14px"><strong>stripPngAlphaChannel</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default true</span><br><span style="font-size:13px">Whether the screenshot's alpha channel should be removed. This means the PNG doesn't support transparancy, but since Minecraft screenshots don't have that it's safe to use.</span><br><br><span style="font-size:14px"><strong>stripScreenshotMetadata</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">If the screenshot metadata should be kept or not. When removes, it lowers the file size slightly.</span><br><br><span style="font-size:14px"><strong>reduceAmountOfColours</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Limits the number of colors used in the image to improve compression. When enabled, fewer colors are used, which reduces the file size but may alter the image&acirc;&euro;&trade;s appearance slightly.</span><br><br><span style="font-size:14px"><strong>reducedColourAmount</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default 256, min 0, max 256</span><br><span style="font-size:13px">The amount of colours that are used when 'reduceAmountOfColours' is enabled.</span><br><br><span style="font-size:14px"><strong>reduceGradient</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Controls the reduction of color gradients to improve compression. When enabled, it simplifies color transitions, either by smoothing or adding noise, to reduce file size while maintaining visual quality.</span></p></div><p style="text-align:center"><br><br><br><img src="https://cdn.serilum.com/description/nameplate/showcase.svg" alt="Showcase" width="838" height="64"></p><p><span style="font-size:14px">A screenshot being saved with compression enabled:</span><br><img src="https://cdn.serilum.com/projects/screenshot-compression/a.png" alt=""></p><p style="text-align:center"><br><br><br><img src="https://cdn.serilum.com/description/nameplate/notes-n-links.svg" alt="Notes and links" width="838" height="64"></p><p><span style="font-size:24px"><strong>You may freely use this mod in any modpack, as long as the download remains hosted within the CurseForge or Modrinth ecosystem.</strong></span><br><br><span style="font-size:18px"><a style="font-size:18px;color:#008000" href="https://serilum.com/" rel="nofollow">Serilum.com</a> contains an overview and more information on all mods available.</span><br><br><span style="font-size:14px">Comments are disabled as I'm unable to keep track of them across so many separate mod pages.</span><span style="font-size:14px"><br>For issues, ideas, suggestions or anything else there is the&nbsp;<a style="font-size:14px;color:#008000" href="https://github.com/Serilum/.issue-tracker" rel="nofollow">Github repo</a>.
