<p><span style="font-size:18px">Random Sheep Colours is a minimalistic configurable mod which gives all new sheep spawns in the world a random fleece (wool) colour. Bred sheeps are not affected. You can set which colours should spawn in the config file. Jeb sheep are also supported and enabled by default.<br><br><strong>Note:</strong> A restart is required for any config changes to take effect.</span></p><p style="text-align:center"><br><br><img
