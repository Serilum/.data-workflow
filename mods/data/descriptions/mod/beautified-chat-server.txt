It is highly configurable and allows users to change the chat input server-wide.&nbsp;The mod variant you're currently looking at just needs the mod in the server mod list. It will make changes to the chat for all connected players without the need to have it on their clients. There is also a timestamp feature to know when messages were sent.<br><br>Version 3.0 adds the ability to create ranks, and set name colours based on those ranks!<br><strong><span style="color:#396;font-size:18px"><br><img src="https://cdn.serilum.com/projects/beautified-chat-server/a.jpg" alt=""><br><br>The client version is also available and is called <a style="font-size:18px;color:#396" href="https://curseforge.com/minecraft/mc-mods/beautified-chat-client" rel="nofollow">Beautified Chat [Client]</a>.</span></strong><br>Both the client and server mod are compatible with each other, server-side takes priority.<br><br><strong><span style="color:#f60;font-size:18px">In default Minecraft, all chat is in the format:</span></strong><br><span style="font-size:14px"><strong>&lt;Username&gt; Message</strong></span><br><br><strong><span style="color:#f60;font-size:18px">The mod by default changes this to:</span></strong><br><span style="font-size:14px"><strong>Timestamp | Username: Message</strong></span><br><br>The default colours can be seen below in the example image. Both the format and the colour codes can be changed with ease via the config. If you don't want the timestamp, just remove it in the&nbsp;<em>chatMessageFormat</em> config option.<br><br>The default timestamp format is 24-hours. If you want AM/PM, just input in the <em>timestampFormat</em> config option "<strong>hh:mm a</strong>" instead of the default "<strong>HH:mm</strong>".</span></p><p style="font-size:14px"><strong>showRankTitles</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default true</span><br><span style="font-size:13px">Whether the rank titles, created in /bcs rank, should be shown before the username.</span><br><br><span style="font-size:14px"><strong>useRankColours</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default true</span><br><span style="font-size:13px">Whether rank colours, set in /bcs rank, should be used as the priority username colour.</span><br><br><span style="font-size:14px"><strong>chatMessageFormat</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default "%timestamp% | %username%: %chatmessage%"</span><br><span style="font-size:13px">Variables: %timestamp% = timestamp set in timestampFormat. %username% = the username of the player who sent the message. %username% = the username of who sent the message.</span><br><br><span style="font-size:14px"><strong>timestampFormat</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default "HH:mm"</span><br><span style="font-size:13px">Example time in formats: 5 seconds past 9 o' clock in the evening. *=Default. *(HH:mm) = 21:00, (HH:mm:ss) = 21:00:05, (hh:mm a) = 9:00 PM</span><br><br><span style="font-size:14px"><strong>rankTitleFormat</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default "[%rank] "</span><br><span style="font-size:13px">The rank format prefixed before %username% when showRankTitles is enabled.</span><br><br><span style="font-size:14px"><strong>chatTimestampColour</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default 8, min 0, max 15</span><br><span style="font-size:13px">The colour of the timestamp in the chat message. The possible values are; 0: black, 1: dark_blue, 2: dark_green, 3: dark_aqua, 4: dark_red, 5: dark_purple, 6: gold, 7: gray, 8: dark_gray, 9: blue, 10: green, 11: aqua, 12: red, 13: light_purple, 14: yellow, 15: white.</span><br><br><span style="font-size:14px"><strong>chatUsernameColour</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default 2, min 0, max 15</span><br><span style="font-size:13px">The colour of the username in the chat messsage.</span><br><br><span style="font-size:14px"><strong>chatMessageColour</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default 15, min 0, max 15</span><br><span style="font-size:13px">The colour of the chat message.</span><br><br><span style="font-size:14px"><strong>chatOtherSymbolsColour</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default 7, min 0, max 15</span><br><span style="font-size:13px">The colour of the other symbols from chatMessageFormat. So everything except the variables.</span></p></div><p style="text-align:center"><br><br><br><img src="https://cdn.serilum.com/description/nameplate/commands.svg" alt="Commands" width="838" height="64"></p><p><span style="font-size:14px">Use <em>/bcs</em> (alias: <em>/beautifiedchatserver</em>). <strong>Needs cheat-access.</strong><br><br><span style="font-size:14px"><em>/bcs rank create &lt;rank-name&gt; &lt;colour&gt;</em> - Create or update a rank with the given chat colour.</span><br><span style="font-size:14px"><em>/bcs rank remove &lt;rank-name&gt;</em> - Delete a rank.</span><br><span style="font-size:14px"><em>/bcs rank list</em> - List all ranks and their colors.</span><br><br><span style="font-size:14px"><em>/bcs player add &lt;player-name&gt; &lt;rank-name&gt;</em> - Assign a rank to a player.</span><br><span style="font-size:14px"><em>/bcs player remove &lt;player-name&gt; &lt;rank-name&gt;</em> - Remove a rank from a player.</span><br><em>/bcs player list</em> - List all players with assigned ranks.</span></p><p style="text-align:center"><br><br><br><img src="https://cdn.serilum.com/description/nameplate/showcase.svg" alt="Showcase" width="838" height="64"></p><p><span style="font-size:14px"><strong>The default format ("%timestamp% | %username%: %chatmessage%"):</strong></span><br><img src="https://cdn.serilum.com/projects/beautified-chat-server/b.jpg" alt="" width="780" height="148"></p><p><br><span style="font-size:14px"><strong>You can also add ranks, and choose colours based on those:</strong></span><br><img src="https://cdn.serilum.com/projects/beautified-chat-server/c.png" alt=""></p><p style="text-align:center"><br><br><br><img src="https://cdn.serilum.com/description/nameplate/notes-n-links.svg" alt="Notes and links" width="838" height="64"></p><p><span style="font-size:24px"><strong>You may freely use this mod in any modpack, as long as the download remains hosted within the CurseForge or Modrinth ecosystem.</strong></span><br><br><span style="font-size:18px"><a style="font-size:18px;color:#008000" href="https://serilum.com/" rel="nofollow">Serilum.com</a> contains an overview and more information on all mods available.</span><br><br><span style="font-size:14px">Comments are disabled as I'm unable to keep track of them across so many separate mod pages.</span><span style="font-size:14px"><br>For issues, ideas, suggestions or anything else there is the&nbsp;<a style="font-size:14px;color:#008000" href="https://github.com/Serilum/.issue-tracker" rel="nofollow">Github repo</a>.
