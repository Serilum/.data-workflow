style="font-size:14px"><strong>followerListHeaderFormat</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default "Followers:"</span><br><span style="font-size:13px">The header true</span><br><span style="font-size:13px">If enabled, shows the follower's health in the GUI list.</span><br><br><span style="font-size:14px"><strong>followerHealthFormat</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default "</span><br><span style="font-size:13px">&lt;health&gt;%"): The format of the health string in the GUI. &lt;health&gt; will be replaced by the style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default " (&lt;distance&gt; blocks</span><br><span style="font-size:13px">The format of the distance string in the GUI. &lt;distance&gt; will be replaced by distance in blocks.</span><br><br></p><p><span style="font-size:14px"><strong>distanceToCheckForFollowersAround</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default 50, min -1, max 300</span><br><span style="font-size:13px">The distance in blocks around the player where the mod checks for tamed, non-sitting followers to add to the list. A value of -1 disables this feature.</span><br><br><span style="font-size:14px"><strong>timeBetweenChecksInSeconds</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default 2, min 0, max 3600</span><br><span style="font-size:13px">The time in seconds in between checking for tamed, non-sitting followers around the player.</span><br><br></p><p><span style="font-size:14px"><strong>followerListPositionIsLeft</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default true</span><br><span style="font-size:13px">Places the follower list on the left.</span><br><br><span style="font-size:14px"><strong>followerListPositionIsCenter</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Places the follower list in the middle.</span><br><br><span style="font-size:14px"><strong>followerListPositionIsRight</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Places the follower list on the right.</span><br><br></p><p><span style="font-size:14px"><strong>followerListHeightOffset</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default 20, min 0, max 3000</span><br><span style="font-size:13px">The vertical offset (y coord) for the follower list. This determines how far down the list should be on the screen. Can be changed to prevent GUIs from overlapping.</span><br><br></p><p><span style="font-size:14px"><strong>drawTextShadow</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default true</span><br><span style="font-size:13px">If the text displayed should have a shadow drawn below it.</span><br><br><span style="font-size:14px"><strong>RGB_R</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default 255, min 0, max 255</span><br><span style="font-size:13px">The red RGB value for the text.</span><br><br><span style="font-size:14px"><strong>RGB_G</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default 255, min 0, max 255</span><br><span style="font-size:13px">The green RGB value for the text.</span><br><br><span style="font-size:14px"><strong>RGB_B</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default 255, min 0, max 255</span><br><span style="font-size:13px">The blue RGB value for the text.</span></p></div><h2><br><br><br><img src="https://cdn.serilum.com/description/nameplate/showcase.svg" alt="Showcase" width="838" height="64"></h2><p><span style="font-size:14px"><strong>The wolf named buddy shown in the GUI:</strong></span><br><img src="https://cdn.serilum.com/projects/gui-followers/a.png" alt="" width="1142" height="641"><br><br></p><p><span style="font-size:14px"><strong>The wolf buddy is currently at 85% health.
