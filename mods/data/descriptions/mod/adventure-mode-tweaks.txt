The idea came from realizing that players are able to break paintings, which obviously shouldn't be the case. The functionality was extended from there.<br><br>For example, without the mod, you are able to break minecarts but not place them. And both break and place boats. Breaking leads can be unintended by RPG map makers as well. Everything is configurable so you can choose exactly what to enable and disable.<br><br>You can also use the mod to have all new players spawn in Adventure Mode. Useful if you have a mod pack that uses a specific scenario. See the features and images below for more information!<br><br><strong><span style="font-size:24px">Features:</span> (while in Adventure Mode)</strong><br>&nbsp;&bull;&nbsp; Everything is entirely configurable.<br>&nbsp;&bull;&nbsp; Force Adventure Mode in new worlds, includes UI updates.<br>&nbsp;&bull;&nbsp; Prevent passive and/or hostile mob attacks.<br>&nbsp;&bull;&nbsp; Prevent users from breaking paintings.<br>&nbsp;&bull;&nbsp; Prevent item frames from being broken, but allow taking items.<br>&nbsp;&bull;&nbsp; Both place and break boats and minecarts (or disable it instead).<br>&nbsp;&bull;&nbsp; Prevent entity interaction entirely.<br>&nbsp;&bull;&nbsp; Prevent breaking and using leads.<br>&nbsp;&bull;&nbsp; Prevent the use of ender pearls.<br>&nbsp;&bull;&nbsp; Disable being able to sleep in beds.<br>&nbsp;&bull;&nbsp; Disable fishing and/or hooking entities.<br>&nbsp;&bull;&nbsp; Prevent shearing sheep and/or milking cows.<br></span><span style="font-size:18px">&nbsp;&bull;&nbsp; Prevent placed TNT from being lit.</span></p><p style="text-align:center"><br><br><img src="https://cdn.serilum.com/description/nameplate/dependencies.svg" alt="Dependencies" width="838" height="64"></p><p><a href="https://curseforge.com/minecraft/mc-mods/collective" target="_blank" rel="nofollow"><img src="https://cdn.serilum.com/description/badges/dependencies/requires/collective.svg" alt="Requires Collective" width="251" height="49"></a><br><img src="https://cdn.serilum.com/description/badges/environment/both.svg" alt="Environment: Client &amp; Server" width="293" height="49"></p><p style="text-align:center"><br><br><br><img src="https://cdn.serilum.com/description/nameplate/configurable.svg" alt="Configurable" width="838" height="64"></p><p><a href="https://serilum.com/configuration-guide" target="_blank" rel="nofollow"><img src="https://cdn.serilum.com/description/badges/configuration-guide.svg" alt="Configuration guide" width="220" height="49"></a></p><p>&nbsp;&nbsp;&nbsp;<span style="font-size:18px">All available config options:</span></p><div class="spoiler" style="margin-left:10px;padding-left:10px"><p><span style="font-size:14px"><strong>forceSurvivalToAdventureMode</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Globally changes Survival Mode to Adventure Mode. This is done when a player joins a world. The Create World Screen UI is updated accordingly as well.</span><br><br><span style="font-size:14px"><strong>preventAttackingCreatureMobs</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Stops players in Adventure Mode from harming passive mobs.</span><br><br><span style="font-size:14px"><strong>preventAttackingMonsterMobs</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Prevents Adventure Mode players from attacking hostile mobs.</span><br><br><span style="font-size:14px"><strong>preventEntityInteraction</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Disables right-click interactions with entities while in Adventure Mode.</span><br><br><span style="font-size:14px"><strong>preventDestroyingPaintings</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default true</span><br><span style="font-size:13px">Adventure Mode players can't break paintings. In vanilla Minecraft this is allowed.</span><br><br><span style="font-size:14px"><strong>preventDestroyingItemFrames</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default true</span><br><span style="font-size:13px">Prevents breaking item frames when in Adventure Mode. In vanilla Minecraft this is allowed.</span><br><br><span style="font-size:14px"><strong>preventLootingItemFrames</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Stops Adventure Mode players from taking items from item frames.</span><br><br><span style="font-size:14px"><strong>preventBreakingMinecarts</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Prevents players in Adventure Mode from destroying minecarts.</span><br><br><span style="font-size:14px"><strong>allowPlacingMinecarts</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default true</span><br><span style="font-size:13px">Allows minecarts to be placed even in Adventure Mode. Default Minecraft behaviour is no placement.</span><br><br><span style="font-size:14px"><strong>preventBreakingBoats</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Prevents players in Adventure Mode from destroying boats.</span><br><br><span style="font-size:14px"><strong>allowPlacingBoats</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default true</span><br><span style="font-size:13px">Allows boat placement while in Adventure Mode. Default Minecraft behaviour is allowed placement.</span><br><br><span style="font-size:14px"><strong>preventCreeperBlockDamage</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default true</span><br><span style="font-size:13px">Stops creepers from destroying blocks when exploded near an Adventure Mode player.</span><br><br><span style="font-size:14px"><strong>preventUseOfEnderPearls</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Blocks ender pearl from being used by Adventure Mode players.</span><br><br><span style="font-size:14px"><strong>preventBedSleeping</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Prevents sleeping in beds while in Adventure Mode.</span><br><br><span style="font-size:14px"><strong>preventLightningTNT</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Prevents Adventure Mode players from lighting TNT with a flint &amp; steel.</span><br><br><span style="font-size:14px"><strong>preventFishing</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Adventure Mode players can't use fishing rods.</span><br><br><span style="font-size:14px"><strong>preventFishingRodHookedEntities</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default true</span><br><span style="font-size:13px">Stops players with fishing rods from hooking entities in Adventure Mode.</span><br><br><span style="font-size:14px"><strong>preventUseOfLeads</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Prevent using and breaking leads in Adventure Mode.</span><br><br><span style="font-size:14px"><strong>preventCowMilking</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Prevents cows from being milked while in Adventure Mode. Redundant if 'preventEntityInteraction' is enabled.</span><br><br><span style="font-size:14px"><strong>preventSheepShearing</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Stops sheep from being sheared by Adventure Mode players. Redundant if 'preventEntityInteraction' is enabled.</span></p></div><p style="text-align:center"><br><br><br><img src="https://cdn.serilum.com/description/nameplate/showcase.svg" alt="Showcase" width="838" height="64"></p><p><span style="font-size:14px"><strong>Be able to force Adventure Mode in the create world screen.
