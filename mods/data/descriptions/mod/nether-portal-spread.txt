By default you can stop the spread by placing four coal blocks within the portal's radius. This can however be changed in the config. Players around the portal will receive a message when the spread effect begins and when it is prevented.<br><br>The mod saves all portal locations to '<em>../world-folder/data/netherportalspread/portals/</em>'. A new portal is detected when it's created or when a player goes from the nether to the overworld. So for old worlds you'll have to go through it and back once to start the spread effect, or re-fire the portal. This makes it a lot more efficient than checking every block in the world for a portal.</span><br><br></p><p><span style="font-size:14px"><em><span style="color:#008000">Version 4.4 and above is compatible with the Immersive Portals mod.</span></em></span></p><p style="text-align:center"><br><br><img src="https://cdn.serilum.com/description/nameplate/dependencies.svg" alt="Dependencies" a message to players around the portal that the nether is spreading and that you can stop the spread with 'preventSpreadBlockAmountNeeded' of the 'preventSpreadBlockString' block.</span><br><br><span style="font-size:14px"><strong>messageOnPortalCreation</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default "<em>You feel a corrupt energy coming from the portal. style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default true</span><br><span style="font-size:13px">When enabled, sends a message to players around the portal that the nether spread has stopped when the portal detects new 'preventSpreadBlockString' blocks.</span><br><br><span style="font-size:14px"><strong>messageOnPreventSpreadBlocksFound</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default "<em>With enough %preventSpreadBlockString% placed, you feel the corrupted energy fade.</em>"</span><br><span style="font-size:13px">The style="font-size:14px"><strong>messageOnPortalBroken</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default "<em>With the nether portal broken, the corrupted energy is no longer able to enter the overworld.</em>"</span><br><span style="font-size:13px">The message sent when a portal is broken.</span><br><br><span style="font-size:14px"><strong>prefixPortalCoordsInMessage</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default true</span><br><span style="font-size:13px">When enabled, shows the portal coordinates in portal messages.</span><br><br></p><p><span style="font-size:14px"><strong>portalSpreadRadius</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default 15, min 1, max 100</span><br><span style="font-size:13px">The radius around the portal to which the nether blocks can spread.</span><br><br><span style="font-size:13px">&nbsp;The delay in ticks in between the spread around the nether portal. 20 ticks = 1 second.</span><br><br><span style="font-size:14px"><strong>instantConvertAmount</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default 50, min 0, max 1000</span><br><span style="font-size:13px">The amount of blocks that are instantly converted to a nether block around a portal when it is detected. If there are existing nether blocks within the radius, their count is subtracted from this number.</span><br><br></p><p><span style="font-size:14px"><strong>preventSpreadWithBlock</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default true</span><br><span style="font-size:13px">When enabled, blocks the spread effect when there are n (defined) prevent-spread-blocks (defined) within the radius.</span><br><br><span style="font-size:14px"><strong>preventSpreadBlockAmountNeeded</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default 4</span><br><span style="font-size:13px">The amount of prevent-spread-blocks (defined) needed within the radius of the nether portal to prevent spread.</span><br><br><span style="font-size:14px"><strong>preventSpreadBlockString</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default "<em>minecraft:coal_block</em>"</span><br><span style="font-size:13px">The block which prevents the nether portal from spreading. By default a coal block (minecraft:coal_block is the namespace ID).</span></p></div><p><br><br><span style="font-size:18px"><strong>By default, the mod will send a message to players around the portal when:</strong></span><br><span style="font-size:14px"><strong>A portal is created, and the spread starts:</strong></span><br><img src="https://cdn.serilum.com/projects/nether-portal-spread/a.jpg" alt="" width="1142" height="533"><br><br><span style="font-size:14px"><strong>Portal spread is stopped with the preventspread blocks (defined in config):</strong></span><br><img src="https://cdn.serilum.com/projects/nether-portal-spread/b.jpg" alt="" width="1142" height="533"><br><br></p><p><span style="font-size:14px"><strong>The message when a portal is broken:</strong></span><br><img src="https://cdn.serilum.com/projects/nether-portal-spread/c.jpg" alt="" width="1142" height="533"><br><br></p><p><span style="font-size:18px"><strong>Configurable spread blocks</strong></span><br>You can define which blocks should spread to what and the chance of it happening in the "<strong>spreadsettings.txt</strong>" file located in "<strong>./config/netherportalspread/</strong>". By default it contains the following:<br><br></p><p><span style="font-size:14px"><em>stone; [netherrack&gt;10 + magma_block&gt;2 + nether_quartz_ore&gt;1],</em></span><br><span style="font-size:14px"><em>grass_block; [netherrack&gt;1],</em></span><br><span style="font-size:14px"><em>dirt; [netherrack&gt;1],</em></span><br><span style="font-size:14px"><em>sand; [soul_sand&gt;1],</em></span><br><span style="font-size:14px"><em>gravel; [soul_sand&gt;1],</em></span><br><span style="font-size:14px"><em>stone_bricks; [nether_bricks&gt;2 + red_nether_bricks&gt;1],<br>sea_lantern; [glowstone&gt;1],<br>farmland; [soul_sand&gt;1],<br>wheat; [nether_wart&gt;1]</em></span></p><p><br><strong>The mod automatically adds "minecraft:" in front of the name values if it contains no ":" (colon).</strong><br><strong>The format is:</strong> FROMBLOCK; [TOBLOCK&gt;WEIGHT (+ TOBLOCK&gt;WEIGHT + TOBLOCK&gt;WEIGHT) ]<br><strong>The weight is automatically converted to chance values. This is calculated by: (1/total_weight)*weight.</strong><br>So the default config file means:<br><br></p><p><span style="font-size:14px"><em>Convert stone into 76.92% netherrack. 15.38% magma blocks
