<p><span style="font-size:18px">Random Mob Effects is a configurable mod created to make the game a little more difficult. Every hostile mob receives a random potion effect upon joining the world. There are a few potion effects disabled by default, but a blacklist is created on first load and you may change the values. The effect applied has a duration of the maximum integer value in java, which is&nbsp;2147483647 ticks; around 30000 hours in-game. The particle effect can be disabled.</span></p><p style="text-align:center"><br><br><img src="https://cdn.serilum.com/description/nameplate/dependencies.svg" alt="Dependencies" width="838" height="64"></p><p><a href="https://curseforge.com/minecraft/mc-mods/collective" target="_blank" rel="nofollow"><img src="https://cdn.serilum.com/description/badges/dependencies/requires/collective.svg" alt="Requires Collective" width="251" height="49"></a><br><img src="https://cdn.serilum.com/description/badges/environment/server.svg" alt="Environment: Server" width="220" height="49"></p><p style="text-align:center"><br><br><br><img src="https://cdn.serilum.com/description/nameplate/configurable.svg" alt="Configurable" width="838" height="64"></p><p><a href="https://serilum.com/configuration-guide" target="_blank" rel="nofollow"><img src="https://cdn.serilum.com/description/badges/configuration-guide.svg" alt="Configuration guide" width="220" height="49"></a></p><p>&nbsp;&nbsp;&nbsp;<span style="font-size:18px">All available config options:</span></p><div class="spoiler" style="margin-left:10px;padding-left:10px"><p><span style="font-size:14px"><strong>potionEffectLevel</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default 1, min 1, max 50</span><br><span style="font-size:13px">The default level of the effects the mod applies, by default 1.</span><br><br><span style="font-size:14px"><strong>hideEffectParticles</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">When enabled, hides the particles from the mobs with an effect.</span><br><br><span style="font-size:14px"><strong>disableCreepers</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default true</span><br><span style="font-size:13px">Creepers can create infinite lingering potion effects which is probably not what you want. When enabled, the mod doesn't give creepers a random effect.</span><br><br></p><p><span style="font-size:14px"><span style="font-size:18px"><strong>The default blacklist</strong></span><br>is generated upon first load of the mod. It cycles through all potion effects within the Forge registry. This means that modded effects are supported, just make sure to refresh the config if a new mod with an effect is added. <em>You can disable an effect by adding an exclamation mark (!) in front of the name.</em><br><span style="font-size:14px">The blacklist file is located in <em><strong>./config/randommobeffects/</strong></em> and is called <em><strong>blacklist.txt</strong></em>.</span><br><img src="https://cdn.serilum.com/projects/random-mob-effects/a.png" alt="" width="1081" height="948"></span></p></div><p style="text-align:center"><br><br><br><img src="https://cdn.serilum.com/description/nameplate/showcase.svg" alt="Showcase" width="838" height="64"></p><p><span style="font-size:16px">Hostile mobs with their random effects:</span></p><div class="spoiler"><p><img src="https://cdn.serilum.com/projects/random-mob-effects/b.webp" alt="" width="1000" height="556"></p></div><p>&nbsp;</p><div class="spoiler"><p><img src="https://cdn.serilum.com/projects/random-mob-effects/c.webp" alt="" width="1000" height="510"></p></div>
