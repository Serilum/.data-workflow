On first load, it generates a config file. You can add items manually, or use the commands in-game. Any items with NBT tags such as enchantments and written books are supported. Modded entities are also supported, just make sure to remove or rename the config file in between adding mods which add custom mobs. If the file does not exist, it'll be generated with all possible entity types.<br><br>The config file generated is located in&nbsp;'<em>./config/configurableextramobdrops/</em>' named&nbsp;'<em>mobdropconfig.txt</em>'. The drop chances are set via the command or can be changed in the config file as well.</span></p><h2><br><br><img src="https://cdn.serilum.com/description/nameplate/dependencies.svg" alt="Dependencies" width="838" height="64"></h2><p><a href="https://curseforge.com/minecraft/mc-mods/collective" target="_blank" rel="nofollow"><img src="https://cdn.serilum.com/description/badges/dependencies/requires/collective.svg" alt="Requires Collective" width="251" height="49"></a><br><a href="https://curseforge.com/minecraft/mc-mods/serilums-customization-bundle" target="_blank" rel="nofollow"><img src="https://cdn.serilum.com/description/badges/dependencies/includes/customization-bundle.svg" alt="Included in Serilum's Customization Bundle" width="416" height="49"></a><br><img src="https://cdn.serilum.com/description/badges/environment/server.svg" alt="Environment: Server" width="220" height="49"></p><h2><br><br><br><img src="https://cdn.serilum.com/description/nameplate/commands.svg" alt="Commands" width="838" height="64"></h2><p><span style="font-size:14px"><span style="font-size:14px"><em>/cemd usage</em></span> - Shows the usage of commands.</span><br><span style="font-size:14px"><span style="font-size:14px"><em>/cemd list</em></span> - Lists all entity names which can be used in <em>addhand</em> and <em>cleardrops</em>.</span><br><span style="font-size:14px"><span style="font-size:14px"><em>/cemd reload</em></span> - Reloads all manual changes to the mobdropconfig file.</span><br><span style="font-size:14px"><span style="font-size:14px"><em>/cemd addhand &lt;entity-name&gt;</em></span> - Adds the item held in the main hand as a drop for the entity named &lt;entity-name&gt; with a 100% dropchance.</span><br><span style="font-size:14px"><span style="font-size:14px"><em>/cemd addhand &lt;entity-name&gt; &lt;drop-chance&gt;</em></span> - Adds the item held in the main hand as a drop for the entity named &lt;entity-name&gt; with a dropchance in between 0 and 1.0.</span><br><span style="font-size:14px"><span style="font-size:14px"><em>/cemd cleardrops &lt;entity-name&gt;</em></span> - Clears all extra drops for the entity named &lt;entity-name&gt;.</span><br><br></p><p><span style="font-size:14px"><em>All commands require cheat-access.</em></span></p><h2><br><br><br><img src="https://cdn.serilum.com/description/nameplate/showcase.svg" alt="Showcase" width="838" height="64"></h2><p><span style="font-size:14px">The <em>/cemd usage</em> command:</span><br><img src="https://cdn.serilum.com/projects/configurable-extra-mob-drops/a.jpg" alt="" width="1000" height="538"><br><br></p><p><span style="font-size:14px">The <em>/cemd list</em> command.
