I needed more basic functionality that was quickly released after each Minecraft update for my <a style="font-size:18px" href="https://curseforge.com/minecraft/modpacks/the-vanilla-experience" rel="nofollow">The Vanilla Experience</a> modpack. It's however highly configurable and might be what you need as well. Intentionally kept simple. If you'd like more control, I recommend using FancyMenu.<br><br><strong>It can:</strong><br>&nbsp;&bull; Set a custom window title and icon.<br>&nbsp;&bull; Replace the Minecraft logo on the main menu.<br>&nbsp;&bull; Set a custom background.<br>&nbsp;&bull; Hide the Minecraft Realms button.<br>&nbsp;&bull; Create a server hosting promo button with a clickable link.<br>&nbsp;&bull; Hide the splash texts next to the logo.<br>&nbsp;&bull; Remove the experimental warning text from mod loaders.<br>&nbsp;&bull; Remove the text in the bottom left.<br></span><br><br></p><p><img style="font-size:18px">All available config options:</span></p><div class="spoiler" style="margin-left:10px;padding-left:10px"><p><span style="font-size:14px"><strong>setCustomWindowTitle</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Whether a custom window Uses the value set in 'customWindowTitle'.</span><br><br><span style="font-size:14px"><strong>customWindowTitle</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default "Custom Window Title"</span><br><span style="font-size:13px">The custom window title that's used when 'setCustomWindowTitle' is enabled.</span><br><br></p><p><span style="font-size:14px"><strong>setCustomWindowIcon</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">If a custom window icon should be set. Two files are needed. Both in PNG format. One a 16 pixel square called 'icon_16x16.png', and one a 32 pixel square called 'icon_32x32.png'. Located in './config/simplemenu/icon'.</span><br><br></p><p><span style="font-size:14px"><strong>replaceMainMenuLogo</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">If a custom main menu logo should be rendered. This will replace the 'Minecraft' text. A 'logo.png' file is needed. Located in './config/simplemenu/logo'. By default the image dimensions are 1024x256. To remove it completely, use an empty .png file.</span><br><br><span style="font-size:14px"><strong>replaceMainMenuEditionLogo</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">If a custom main menu 'edition' logo should be rendered. This will replace the 'Java Edition' text. An 'edition.png' file is needed. Located in './config/simplemenu/logo'. By default the image dimensions are 512x64. To remove it completely, use an empty .png file.</span><br><br></p><p><span style="font-size:14px"><strong>setCustomBackground</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">If a custom background should be rendered. A 'background.png' file is needed. Located in './config/simplemenu/background/'.</span><br><br></p><p><span style="font-size:14px"><strong>hideMinecraftRealmsButton</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">If the Minecraft Realms button should be hidden from the main menu.</span><br><br><span style="font-size:14px"><strong>expandModsButton</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Whether the Mods button's width should be expanded to match the singleplayer button width. Can be useful with 'hideMinecraftRealmsButton' disabled. Only applicable for Forge and NeoForge.</span><br><br><span style="font-size:14px"><strong>enableServerPromoButton</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Whether the Minecraft Realms button should be replaced by a serverhosting promo button.</span><br><br><span style="font-size:14px"><strong>serverPromoButtonTextDefault</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default "Need a Server?"</span><br><span style="font-size:13px">The default server promo button text.</span><br><br><span style="font-size:14px"><strong>serverPromoButtonTextOnHover</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default "Use code PROMO for 25% off"</span><br><span style="font-size:13px">The server promo button text shown when hovering it.</span><br><br><span style="font-size:14px"><strong>serverPromoButtonClickURL</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default "https://localhost/exampleurl"</span><br><span style="font-size:13px">The URL that's being navigated to when clicking the server promo button.</span><br><br></p><p><span style="font-size:14px"><strong>hideSplashText</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Whether the splash text should be hidden on the main menu.</span><br><br><span style="font-size:14px"><strong>removeExperimentalModLoaderText</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Whether experimental/beta mod loader text should be removed from the main menu.</span><br><br><span style="font-size:14px"><strong>removeTextBottomLeft</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">Whether the text in the bottom left should be removed from the main menu.</span></p></div><p
