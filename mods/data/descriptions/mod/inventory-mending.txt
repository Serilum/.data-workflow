<p><span style="font-size:18px">Inventory Mending is a quality of life mod that takes away the need to hold the tools with mending you'd like to repair. Any item inside the inventory that's enchanted with mending is able to be repaired. There is also a config option available to only repair items in the toolbar.<br><br>The mod allows the default behaviour to run first, which is repairing the held/equipped items. If those are repaired, it checks for other items inside the inventory.</span></p><p style="text-align:center"><br><br><img src="https://cdn.serilum.com/description/nameplate/dependencies.svg"
