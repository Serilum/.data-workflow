If a player has cheat-access, they can reload the inventory via the '<em>/deathbackup load &lt;index&gt;</em>' command. '<em>/deathbackup list</em>' shows all the latest deaths with indices which can be retrieved. The inventories are saved in a folder with the players' name, so it works in server environments as well as singleplayer.</span></p><h2><br><br><img
