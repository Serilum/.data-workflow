It's highly configurable. By default, whenever a player walks around at night, any flower can be a mimic and turn into a mob. When this happens, lightning will strike. The mimic will have a custom name. When the player kills the mimics, extra items and experience are dropped. The flower from which it transforms, will be placed back in the original location. There is a <strong>1/50</strong> chance that a flower is a mimic.<br><br><strong>Can cause some jump scares! <span style="font-size:18px">👻</span></strong></span><span style="font-size:18px"><br><br>While there are default values for an instant experience, it's possible to change a lot of behaviour. Which flowers can become mimics, what items are dropped and various settings when and how mimics should appear.</span></p><p style="text-align:center"><br><br><img style="font-size:14px"><strong>flowerIsMimicChance</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default 0.02, min 0, max 1.0</span><br><span style="font-size:13px">The chance an encountered flower is a mimic, and transforms into a mob.</span><br><br><span style="font-size:14px"><strong>extraExperienceOnMimicDeath</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default 30</span><br><span style="font-size:13px">The amount of extra experience that should drop whenever a mimic is killed.</span><br><br><span style="font-size:14px"><strong>dropExtraItemsOnMimicDeath</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default true</span><br><span style="font-size:13px">Whether the items specified in ./config/flowermimics/.. should be dropped whenever a mimic is killed.</span><br><br><span style="font-size:14px"><strong>onlyTransformMimicsAtNight</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default true</span><br><span style="font-size:13px">Whether flowers should only transform into mobs when it is night time. When disabled, they can always transform.</span><br><br><span style="font-size:14px"><strong>resetMimicsBeforeNightTime</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default true</span><br><span style="font-size:13px">Whether mimic flower possibilities should be reset when the day turns into night.</span><br><br><span style="font-size:14px"><strong>removeFlowerBlockOnMimicSpawn</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default true</span><br><span style="font-size:13px">Whether the flower block from which the mimic transforms should be removed on spawn.</span><br><br><span style="font-size:14px"><strong>dropFlowerItemOnMimicDeath</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">If the mimic flower should be dropped on death. Must have 'removeFlowerBlockOnMimicSpawn' enabled.</span><br><br><span style="font-size:14px"><strong>replaceOriginalFlowerBlockOnMimicDeath</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default true</span><br><span style="font-size:13px">If the original flower block should be replaced when a mimic dies. Must have 'removeFlowerBlockOnMimicSpawn' enabled. Destination must have an air block.</span><br><br><span style="font-size:14px"><strong>placeFlowerBlockWhereMimicDies</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default false</span><br><span style="font-size:13px">If the a flower block should be placed where a mimic dies. Must have 'removeFlowerBlockOnMimicSpawn' enabled. Destination must have an air block.</span><br><br><span style="font-size:14px"><strong>showLightningOnMimicSpawn</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default true</span><br><span style="font-size:13px">If lightning should be shown whenever a mimic spawns.</span><br><br><span style="font-size:14px"><strong>showLightningOnFlowerBlockReturn</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default true</span><br><span style="font-size:13px">If lightning should be shown if a flower block is re-placed into the world.</span><br><br><span style="font-size:14px"><strong>preventMimicsOnPlacedAndBoneMealFlowers</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default true</span><br><span style="font-size:13px">Whether manually placed and bone mealed flowers should not spawn mimics.</span><br><br><span style="font-size:14px"><strong>checkForMimicAroundBlockRange</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default 15, min 1, max 30</span><br><span style="font-size:13px">How many blocks away from the player the mod should check for mimics. Increasing this will take more processing power.</span><br><br><span style="font-size:14px"><strong>mimicTransformAroundPlayerRange</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default 5, min 1, max 30</span><br><span style="font-size:13px">How close the player should be to a mimic flower in order for it to transform.</span><br><br><span style="font-size:14px"><strong>checkForMimicFlowersDelayInTicks</strong></span><span style="font-size:12px;color:#8a8d91">&nbsp;&nbsp;default 50, min 1, max 3600000</span><br><span style="font-size:13px">How often the mod should check for flower mimics around the player. 20 ticks = 1 second</span><br><br></p><p><span style="font-size:18px">There are also two config files located in <em><strong>./config/flowermimics/. :</strong></em></span><br><br></p><p><span style="font-size:14px"><strong><span style="font-size:18px">It's possible to configure what flower mimics will turn into.
