Instead of only having a purpur coloured shell, they'll have one of each of the 16 colours. It's also possible for them to spawn as the normal variant without a colour. You can configure the list of colours used. So if you'd for example only want to make them red that is also possible.<br><br>The shulker shell dropped is still the default one.
