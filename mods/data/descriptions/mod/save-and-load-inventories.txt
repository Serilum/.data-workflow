<p><span style="font-size:18px">Save and Load Inventories adds commands which allow the user to easily save their current inventory to a file, load an inventory from a file and list the current saved inventories. Useful for administrators or event servers. NBT tags are supported, any things like written books or enchantments won't be a problem. Players need cheat access to use the commands.<br><br></span><span style="font-size:18px">Th</span><span style="font-size:18px">e inventories are saved in ./config/saveandloadinventories/*.txt.</span></p><p style="text-align:center"><br><br><img src="https://cdn.serilum.com/description/nameplate/dependencies.svg" alt="Dependencies" width="838" height="64"></p><p><a href="https://curseforge.com/minecraft/mc-mods/collective" target="_blank" rel="nofollow"><img src="https://cdn.serilum.com/description/badges/dependencies/requires/collective.svg" alt="Requires Collective" width="251" height="49"></a><br><a href="https://curseforge.com/minecraft/mc-mods/serilums-customization-bundle" target="_blank" rel="nofollow"><img src="https://cdn.serilum.com/description/badges/dependencies/includes/customization-bundle.svg" alt="Included in Serilum's Customization Bundle" width="416" height="49"></a><br><img src="https://cdn.serilum.com/description/badges/environment/server.svg" alt="Environment: Server" width="220" height="49"></p><p style="text-align:center"><br><br><br><img src="https://cdn.serilum.com/description/nameplate/commands.svg" alt="Commands" width="838" height="64"></p><p><span style="font-size:14px"><em>/saveinventory &lt;inventory_name&gt;</em> - Saves the current inventory of the user as &lt;inventory_name&gt;.</span><br>&nbsp; alias: <em>/si &lt;inventory_name&gt;</em><br><span style="font-size:14px"><em>/loadinventory &lt;inventory_name&gt;</em> - Loads an inventory named &lt;inventory_name&gt; previously saved.</span><br>&nbsp; alias: <em>/li &lt;inventory_name&gt;</em><br><span style="font-size:14px"><em>/loadinventory &lt;inventory_name&gt; &lt;player_name&gt;&nbsp;</em>- Loads an inventory to the target player named &lt;player_name&gt;<br><em><span style="font-size:12px">&nbsp; alias: /li &lt;inventory_name&gt; &lt;player_name&gt;</span><br>/listinventories</em> - Lists all saved inventories.</span></p><p style="text-align:center"><br><br><br><img src="https://cdn.serilum.com/description/nameplate/showcase.svg" alt="Showcase" width="838" height="64"></p><p><span style="font-size:14px">Save the current inventory with the name 'diamond' (<em>/saveinventory diamond</em>):</span><br><img src="https://cdn.serilum.com/projects/save-and-load-inventories/a.jpg" alt="" width="1142" height="591"><br><br></p><p><span style="font-size:14px">And when loading it (<em>/loadinventory diamond</em>):</span></p><div class="spoiler"><p><img src="https://cdn.serilum.com/projects/save-and-load-inventories/d.webp" alt="" width="1000" height="514"></p></div><p>&nbsp;<br><span style="font-size:14px">Save the current inventory as 'food' with the shortened alias command (<em>/si food</em>):</span><br><img src="https://cdn.serilum.com/projects/save-and-load-inventories/c.jpg" alt="" width="1142" height="591"><br><br></p><p><span style="font-size:14px">And loading it with the shortened command (<em>/li food</em>):</span></p><div class="spoiler"><p><img src="https://i.gyazo.com/a5e6892d90fc6b7d21cfc9e5331093e8.gif" alt=""></p></div><p>&nbsp;</p><p><br><span style="font-size:14px"><strong>Here's an example of setting the inventory of another player:</strong></span><br><img src="https://cdn.serilum.com/projects/save-and-load-inventories/e.jpg" alt="" width="1142" height="1093"></p>
