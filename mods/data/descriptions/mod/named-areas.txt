<p><span style="font-size:25px"><strong><span style="font-size:25px">&nbsp; &nbsp; &nbsp;This mod is no longer actively being developed because it has been superseded by <a style="color:#008000;font-size:25px" href="https://curseforge.com/minecraft/mc-mods/areas" rel="nofollow">Areas</a>, which works in multiplayer!</span><br><br></strong><span style="font-size:14px;color:#f00">This mod only works in singleplayer.</span><strong><br></strong>Named Areas is a minimalistic configurable mod which allows the creation of custom named areas/towns with a radius or by biome using signs with messages via GUI or chat. Any player entering or leaving the area will receive a message, which is configurable in the config. Overlap of areas is allowed.</span><br><br><strong><span style="text-decoration:underline"><span style="font-size:14px">Configurable</span><br></span></strong></p><div class="spoiler"><p><span style="font-size:14px"><strong>__sendChatMessages</strong></span>&nbsp;(default = false):&nbsp;<br>When enabled, sends the player the area notifications in chat.<br><span style="font-size:14px"><strong>__showHUDMessages</strong></span>&nbsp;(default = true):&nbsp;<br>When enabled, sends the player the area notifications in the HUD on screen.<br><br><span style="font-size:14px"><strong>_HUDOnlyAreaName</strong></span>&nbsp;(default = false):&nbsp;<br>When enabled, only shows the areaname in the HUD. When disabled, the prefixes and suffices will also be used.<br><span style="font-size:14px"><strong>_HUDMessageFadeDelayMs</strong></span>&nbsp;(default = 3500, min 100, max 360000):&nbsp;<br>The delay in ms after which the HUD message should fade out.<br><span style="font-size:14px"><strong>_HUDMessageHeightOffset</strong></span>&nbsp;(default = 10, min 0, max 3000):&nbsp;<br>The vertical offset (y coord) for the HUD message. This determines how far down the message should be on the screen. Can be changed to prevent GUIs from overlapping.<br><br><span style="font-size:14px"><strong>_HUD_RGB_R</strong></span>&nbsp;(default = 100, min 0, max 255):&nbsp;<br>The red RGB value for the HUD message.<br><span style="font-size:14px"><strong>_HUD_RGB_G</strong></span>&nbsp;(default = 200, min 0, max 255):&nbsp;<br>The green RGB value for the HUD message.<br><span style="font-size:14px"><strong>_HUD_RGB_B</strong></span>&nbsp;(default = 50, min 0, max 255):&nbsp;<br>The blue RGB value for the HUD message.<strong><span style="text-decoration:underline"><br></span><br></strong><span style="font-size:14px"><strong>joinPrefix</strong></span>&nbsp;(default = "Entering "): <br>The prefix of the message whenever a player enters an area.<br><strong><span style="font-size:14px">joinSuffix</span>&nbsp;</strong>(default = "."): <br>The suffix of the message whenever a player enters an area.<strong><br></strong><span style="font-size:14px"><strong>leavePrefix</strong></span>&nbsp;(default = "Leaving "): <br>The prefix of the message whenever a player leaves an area.<br><strong><span style="font-size:14px">leaveSuffix</span>&nbsp;</strong>(default = "."): <br>The suffix of the message whenever a player leaves an area.<br><strong><br><span style="font-size:14px">biomeJoinPrefix</span></strong>&nbsp;(default = "Entering a biome named "): <br>The prefix of the message whenever a player enters a named biome area.<br><strong><span style="font-size:14px">biomeJoinSuffix</span>&nbsp;</strong>(default = "."): <br>The suffix of the message whenever a player enters a named biome area.<strong><br></strong><span style="font-size:14px"><strong>biomeLeavePrefix</strong></span>&nbsp;(default = "Leaving a biome named "): <br>The prefix of the message whenever a player leaves a named biome area.<br><strong><span style="font-size:14px">biomeLeaveSuffix</span>&nbsp;</strong>(default = "."): <br>The suffix of the message whenever a player leaves a named biome area.</p></div><p>&nbsp;<br><br><strong><span style="font-size:18px">How it works:</span></strong><strong><br>The first three lines are part of the area name.
