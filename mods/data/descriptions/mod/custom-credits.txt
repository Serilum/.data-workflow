On first load, the default files are copied to the <em><strong>./config/customcredits/</strong></em> folder. Here you can add or change what is being displayed. Made to easily append to the current credits. While it's possible to replace it entirely, I'd recommend to leave the current credits as is and add your text to the top.<br><br>A <em><strong>/credits show</strong></em> command is also added to easily see any changes made. A reboot is not required, as the files are being read whenever the credits screen is shown.</span></p><h2><br><br><img src="https://cdn.serilum.com/description/nameplate/dependencies.svg"
