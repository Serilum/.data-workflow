Without the mod, the output is a long list of all white text and not sorted alphabetically. When users try to use the command with more mods installed, the list might even be longer than the maximum amount of text which can be shown. The result is loss of information.<br><br>With the mod, the long list of commands is sorted and split into pages. Via <em>/help &lt;page&gt;</em> you can go through all commands, which are also more easily readable via different text colours. Those colours and the amount of commands shown per page can be changed in the config. See below for example images.<br><br>Also works for the specific information via <em>/help &lt;command&gt;</em>.</span></p><p style="text-align:center"><br><br><img
